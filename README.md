# XにXが爆誕するやつ

　ブラウザ版XのUIを改造する+αのchrome拡張機能です。他のX上で機能する拡張機能と同時に動かした場合、意図しない挙動をする場合があります。<br>
　google chrome ver.142.0.7444.176で動作を確認しています。他のブラウザでは動作を確認していません。<br>
　現在の機能は以下5つです。<br>

## 1. Loading中のXロゴをルギアに置換

　Xにアクセスした際のロード中の画面に表示されるXロゴがルギアになります。かわいいね。<br>
　一度正規のロゴが表示されるのは正常な挙動です。綺麗に置換できる方法をご存じの方は是非おせーて。<br>

## 2. 左上のXロゴをホバーアクション付きのルギアに置換

　マウスカーソルをホバーさせるとルギアが振り向いてくれます。かわいいね。<br>

## 3. 『ポストする』ボタンの文字置換 + ルギアが嘶くアニメーション

　『ポストする』という文字列を『ギャーアアス』に置換します。また、ポストした際に左上のルギアが叫んでくれます。かわいいね。<br>
　音声を追加することもできます。詳しくは後述。<br>

## 4. 新規通知を取得した際にルギアが飛ぶ

　いいねやリツイート、リプライなどの通知を受け取った際、画面上をルギアが高速で駆け抜けます。かわいいね。<br>
　尚、若干ですが画面を振動させる演出も含んでいますので画面酔いにお気を付けください。<br>
　音声を追加することもできます。詳しくは後述。<br>

## 5. ファビコンをルギアに置換

　タブに表示されるXロゴもルギアになります。かわいいね。<br>

## おまけ

　ポストの詳細を表示した際の"Grokにこのポストを説明してもらう"のポップアップが出なくなります。Grok自体は使えます。<br>


# 導入方法
1. 本拡張機能をPCにダウンロードする<br>
2. chromeのデベロッパーモードをオンにする<br>
3. "パッケージ化されていない拡張機能を読み込む"をクリック<br>
4. srcフォルダを選択して読み込ませる<br>


# 音声の追加方法

　機能3.と機能4.の動作には音声を追加することができます。<br>

1. 音声ファイルをそれぞれ用意する（☆おすすめ☆　3→ルギアの鳴き声　4→3世代そらをとぶのSE）<br>
2. 音声ファイルをsoundフォルダに入れる<br>
3. file.jsonをテキストエディタで開き、3の音声ファイルのファイル名（e.g.:lugia.wav）を"roar-sound"の右側のダブルクォーテーションの中に、5の音声ファイルのファイル名（e.g.:fly.wav）を"flying-sound"の右側のダブルクォーテーションの中に入力して保存する<br>
4. （必要に応じて）拡張機能を再読み込みし、Xのページも再読み込みする<br>
<br>
※お好きな音声を追加することができますが、鳴らすタイミングは固定されています。微調整はReaper等の音源編集ツールで頑張ってください。<br>
